### 项目简介

本项目后端使用开源项目 NeteaseCloudMusicApi (https://github.com/Binaryify/NeteaseCloudMusicApi)，仅供学习使用，无侵权行为

前端采用Vue2+element ui  ,是一个拥有登录注册，音乐推荐，音乐，歌手，排行榜，专辑搜索，音乐播放等功能的一个音乐播放平台。
